# Aeterna Codex Prompt Bundle

This folder contains the full ordered prompt system for building the Aeterna Luxe Wellness website.

## Use order

1. Read `docs/README_BUILD_ORDER.md`.
2. Open `docs/CODEX_PROMPTS_ALL_IN_ONE.md` if you want one master prompt file.
3. Or run the prompts individually from `docs/prompts/00_MASTER_PROMPT.md` through `docs/prompts/12_FINAL_QA_DEPLOY.md`.
